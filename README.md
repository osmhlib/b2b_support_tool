# B2B Support Tool

B2B Support Tool - це Windows-застосунок для спеціалістів технічної підтримки. Програма об'єднує типові сервісні дії: очищення системи, мережеву діагностику, перезапуск модулів, оновлення ПЗ, запуск інструментів відновлення Windows та додаткові адміністративні операції.

## Вимоги до середовища

- ОС: Windows 10 або новіша.
- Платформа: Windows x64.
- Framework: .NET 10.0 Windows.
- Інтерфейс: WPF.
- Рекомендований запуск: від імені адміністратора.

Для збірки з вихідного коду потрібен .NET SDK 10.0 або новіший.

Якщо програма опублікована як self-contained executable, встановлювати .NET Runtime на клієнтській машині не потрібно. Для framework-dependent запуску потрібен .NET Desktop Runtime 10.0.

## Збірка

У PowerShell перейдіть у папку проєкту:

```powershell
cd D:\dev\b2b_support_tool
```

Виконайте збірку:

```powershell
dotnet build
```

Для створення self-contained релізної версії:

```powershell
release.bat
```

Готовий виконуваний файл буде створено в папці:

```text
bin\Release\net10.0-windows\win-x64\publish\
```

## Запуск

Рекомендований спосіб запуску:

1. Відкрити папку з опублікованим `.exe` файлом.
2. Натиснути правою кнопкою миші.
3. Обрати `Run as administrator`.

Адміністративні права потрібні для операцій із системними службами, папками Windows, реєстрацією DLL та repair-командами.

## Використання

Інтерфейс програми поділений на вкладки. У нижній частині вікна розташовано поле виводу результатів.

Під час виконання операції:

- кнопки тимчасово блокуються;
- у заголовку з'являється `[BUSY]`;
- результат виводиться у вікно програми;
- повідомлення записуються у log-файл.

### Clean

Очищення системи:

- видалення check-файлів старіших за 14 днів;
- видалення log-файлів старіших за 14 днів;
- очищення `C:\Windows\Temp`;
- очищення кошика Windows.

### Ping/trace

Мережева діагностика:

- `ping ftp.base2base.com.ua`;
- `tracert ftp.base2base.com.ua`;
- `ping crm.base2base.com.ua`;
- `tracert crm.base2base.com.ua`.

### Modules

Перезапуск модулів:

- ShopdeskIndicator;
- Bridge;
- TccLocalProcessor.

Програма завершує відповідний процес, очікує 2 секунди та запускає модуль повторно.

### Update

Встановлення або оновлення:

- Shopdesk;
- ScaleServer.

Якщо обрано обидва пункти, інсталятори запускаються послідовно: спочатку Shopdesk, потім ScaleServer.

### Repair

Запуск системних інструментів Windows:

- `sfc /scannow`;
- `DISM /Online /Cleanup-Image /RestoreHealth`;
- `chkdsk C: /f /r`.

### Other

Додаткові дії:

- перезапуск служби друку;
- оновлення AnyDesk ID;
- реєстрація бібліотек для вагового обладнання;
- внутрішня сервісна дія активації Windows.

## Логування

Лог записується у файл:

```text
support_tool.log
```

Файл створюється поруч із виконуваним файлом програми.

Формат запису:

```text
[yyyy-MM-dd HH:mm:ss] message
```

Якщо розмір log-файлу перевищує 5 MB, він очищується під час наступного запуску програми.

## Типові проблеми

Якщо операція завершується помилкою доступу:

- перезапустіть програму від імені адміністратора;
- перевірте обмеження антивірусу або політик Windows.

Якщо модуль не запускається:

- перевірте наявність `.exe` файлу за очікуваним шляхом;
- переконайтесь, що назва процесу відповідає модулю.

Якщо оновлення не виконується:

- перевірте доступ до інтернету;
- переконайтесь, що інсталятор не заблокований системою безпеки.

Якщо після натискання `START` нічого не виконується:

- переконайтесь, що обрано хоча б один чекбокс.

## Застереження

Програма виконує адміністративні дії та може видаляти файли із системних і прикладних папок. Перед запуском операції потрібно переконатися, що вона відповідає поточному сценарію підтримки.
